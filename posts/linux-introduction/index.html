Linux Introduction

February 8, 2018 2-minute read
marthinpsrb
linux • technology

Linux is a computer operating system. An operating system consists of the software that manages your computer and lets you run application on it. The features that make up Linux and similar computer operating systems include the following.

  1. Detecting and preparing hardware

When you turn on your computer, the Linux system will boot up and loads the software (drivers and modules) needed to access those particular hardware devices.

  1. Managing Processes

Linux have role to decide which have access to the CPU and when. The system also must provide ways to starting, stopping, and changing the status of process

  1. Managing memory

and swap space (extended memory) must be allocated to application as they need memory. The operating system handles how requests for memory.

  1. Providing user interfaces

The operating systems must provide ways of accessing the system. The first Linux systems were accessed from a command line interpreter called a shell. Today graphical desktop interfaces are commonly available as well.

  1. Controlling file systems

Filesystem structures are built into the operating system. The operating systems controls ownership of and access to the files and directories/folders that the filesystems contain.

  1. Providing user access and authentication

Creating user accounts and allowing boundaries to be set between users is a basic feature of Linux. Separate user and group accounts enable users to control their own files and process.

  1. Offering administrative utilities

Linux, hundreds(perhaps thousands) of commands and graphical windows are available to do such things as add users, manage disks, monitor the network, install software, and generally secure and manage your computer

  1. Starting up services

To use printers, handle log messages and provide a variety of system and network services, processes called daemon processes run in the background waiting for request to come in.

  1. Programming tools

A wide variety of programming utilities for creating applications and libraries for implementing specialty interfaces are available with Linux.
